make menuconfig

By Sam
March 3, 2021

While trying to run the make menuconfig to configure the build of Linux Kernel I encountered some issues. Mainly that I didn't have many of the packages installed. 

This is just a list of commands, i.e package names to install. It might differ based on what you already have installed on your computer. Its in the order the errors were thrown out saying something wasn't available in order to use make menuconfig

sudo apt-get install pkg-config

To install ncurses a sort of terminal gui. 

sudo apt-get install libncurses5-dev libncursesw5-dev

If flex not found

sudo apt-get install flex

If Bison not found

sudo apt-get install bison

I could run make menuconfigafter installing those
